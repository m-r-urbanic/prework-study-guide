Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title is the tab description in your browser.
  • Line Item elements are bullet points.
  • The name of a semantic HTML element describes what the element does.
  • Some elements, such as 'div' are not semantic.
  • Content should have an opening and closing tag.
  • Using proper header order and providing alt text is important for screen readers.

CSS

  • CSS means 'Cascading Style Sheets'
  • The three types of CSS are inline CSS, internal CSS, and external CSS.
  • Inline applies to a specific subsection of HTML.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: will pull the main branch.
  • git add -A comitts all files in the working branch
  • git commit -m "your message here": will commit your code with the provided message to the repo.
  • Git provides version control for projects.
  • Changes are not part of the main project until they are merged to master.
